🎉 Groceezy – The Ultimate Grocery Shopping Experience

Groceezy is an advanced and user-friendly online grocery shopping platform designed using the MERN stack (MongoDB, Express.js, React.js, Node.js). With Groceezy, users can easily browse, shop, and purchase their grocery needs online in a simple and efficient manner. The app ensures smooth performance, a modern design, and an engaging user experience, making it your go-to platform for all grocery shopping requirements.

==================================================================================

🚀 Features

🛒 Browse Grocery Categories – Explore various grocery categories including fresh produce, dairy, snacks, beverages, and more, all neatly organized for easy shopping.

⚡ Fast and Responsive – The combination of React.js and Node.js ensures fast page loads and smooth navigation for an outstanding shopping experience.

🔍 Smart Search and Filters – Easily search for your desired products using an intuitive search bar and apply filters for price range, category, and more.

💳 Secure Payment Integration – Groceezy uses secure payment gateways, providing users with safe and encrypted transaction methods.

📱 Fully Responsive Design – Developed using Tailwind CSS, Groceezy adapts seamlessly to different screen sizes, from desktop to mobile.

🎨 Sleek and Modern Interface – The platform boasts a modern, clean UI, making the shopping experience visually appealing and user-friendly.

🔄 Real-Time Data Sync – Groceezy provides real-time product data updates, ensuring that your product catalog is always fresh and accurate.

==================================================================================

🛠️ Tech Stack

🖥️ Frontend:

React.js – For dynamic and real-time updates to the user interface.

Tailwind CSS – For responsive design and modern UI components.

⚙️ Backend:

Node.js – For server-side operations and API request handling.

Express.js – For building a robust backend to handle HTTP requests and routing.

🗃️ Database:

MongoDB – A NoSQL database that stores product listings, user data, and orders efficiently.

🔒 Authentication:

JWT (JSON Web Tokens) – Ensures secure authentication for user login and session management.

==================================================================================

🔧 Installed Dependencies

• Tailwind CSS → npm install tailwindcss @tailwindcss/vite

• React Icons → npm install react-icons (For displaying icons in the application)

• Axios → npm install axios (For making HTTP requests to the backend API)

• React Router DOM → npm install react-router-dom (For managing client-side routing) 

• Express.js → npm install express (Backend routing and handling requests)

• MongoDB (Mongoose) → npm install mongoose (For database management)

• JWT → npm install jsonwebtoken (For handling secure user authentication)

▶️ How to Run the Groceezy Project

🟢 Install Node.js

Download the latest LTS version from Node.js. Verify the installation with node -v and npm -v.

📂 Navigate to the Project Directory

Open a terminal and run cd /path/to/project to go to the Groceezy project folder.

🗑️ Delete Existing node_modules (if necessary)

If you have any issues with the dependencies, remove the node_modules folder by running rm -rf node_modules.

🔄 Reinstall Dependencies

Run npm install to reinstall all the required dependencies.

🚀 Run Development Servers

For the frontend, use npm run dev (The frontend will be available at localhost:3000).

For the backend, use npm run server (The backend will be available at localhost:5000).
